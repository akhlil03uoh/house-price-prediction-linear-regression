# House Price Prediction using Linear Regression

This project predicts house prices using the California Housing dataset.
It demonstrates the complete machine learning workflow:
data loading, EDA, preprocessing, training, and evaluation.

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Dataset
California Housing Dataset (Scikit-learn)

## Output
MAE, RMSE, R2 Score, and visualizations.
